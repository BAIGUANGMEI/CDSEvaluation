我想开发一个课程评价系统，用户可以注册登录后，对已有的课程进行打分和评价，管理员用户可以创建用于评价的课程。

功能：

课程：
课程编号（必填）
课程名称（必填）
课程教授（选填）
按不同专业划分例如ECIC，CS，FADT，STAT

课程评价：
课程编号（必填）
课程名称（必填）
课程教授（选填）
课程总体评分（1-10）
课程授课难度（1-10）
课程作业难度（1-10）
课程是否有课内测试（Y/N）
课程是否有期末考试（Y/N）
课程考试难度（1-10）
课程是否考勤（Y/N）
其他（文本）

用户：
可以在任意课程下新增自己的评价
可以在个人主页看见自己所有的评价

管理员用户：
在用户的功能基础上可以删除或者修改用户的评价内容
可以增加删除修改课程的信息

技术栈：
前端使用VUE3
后端使用Python
部署采用cloudflare，前端使用Pages，后端使用Worker，数据库使用D1 SQL
